  2. Difference between class and IDs ?
    3. class
    The HTML class attribute specifies one or more class names for an element. Classes are used by CSS and JavaScript to select and access specific elements.
    ids
    The id attribute specifies a unique id for an HTML element. The value of the id attribute must be unique within the HTML document.
  3. What are self closing tags ?
    4. self closing tag
    A self-closing tag in HTML is a kind of HTML tag that does not need to be closed manually by its closing tag, which means it does not have a separate closing tag as /tag. Some few self-closing tags are input/, hr/, br/, img/, etc
  4. How many types of headings are there ?How to decide which to use where ?
    5. heading tag
    HTML defines six levels of headings, and these heading elements are H1, H2, H3, H4, H5, and H6. The H1 element is the highest or most important level, and the H6 element is the least important.
